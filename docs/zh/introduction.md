# 项目介绍

NekoImageGallery是一个基于Clip模型和Qdrant向量数据库的在线AI图片搜索引擎。它支持关键词搜索和以图搜图。

## ✨ 功能特性

- 使用Clip模型为每张图片生成768维向量作为搜索基础。无需手动标注或分类，分类类别不限。
- 支持OCR文字搜索，使用PaddleOCR从图片中提取文字，并使用BERT生成文字向量进行搜索。
- 使用Qdrant向量数据库进行高效的向量搜索。
